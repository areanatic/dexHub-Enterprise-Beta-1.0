# Brainstorming Session Validation Checklist

## Pre-Session
- [ ] Topic clearly defined
- [ ] Session goals stated
- [ ] Appropriate brainstorming technique selected
- [ ] Time constraints communicated

## During Session
- [ ] All ideas captured without judgment
- [ ] Multiple techniques applied (minimum 2)
- [ ] Ideas grouped into themes
- [ ] Participants encouraged to build on ideas

## Post-Session
- [ ] Ideas categorized and prioritized
- [ ] Key themes identified
- [ ] Top ideas selected for further development
- [ ] Action items defined with owners
- [ ] Session results saved to configured output folder
- [ ] Follow-up timeline established

## Quality Gates
- [ ] Minimum 10 ideas generated
- [ ] At least 3 distinct themes identified
- [ ] Top 3-5 ideas have clear next steps
- [ ] Session document follows template format
